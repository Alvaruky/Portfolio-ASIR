Autenticación mediante usuarios y contraseñas

Para realizar la autenticación de usuarios y contraseñas deberemos de crear los usuarios "Bart" y "lisa", para ello deberemos de meternos en vlan en el switch.
IOS ofrefrece la posibilidad de autenticación por usuario, donde cada uno de ellos dispondrá de sus propias credenciales, eliminando de esta manera la contraseña compartida.


Configuración de la Autenticación (2 técnicas):

El switch genera una base de datos en el propio dispositivo que almacena tanto los usuarios como las contraseñas definidas. Para realizar la configuración primero de todo deberemos de crear los usuarios necesarios mediante la ejecución del comando "username [nombreUsuario] password"
