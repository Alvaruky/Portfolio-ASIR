Gestión de Bases de Datos

Contar el numero de alumnos que están en una empresa

            
        SELECT E2.NOMBRE AS EMPRESA, COUNT(*) AS "Número de alumnos"
        FROM ALUMNOS A
        JOIN EMPRESAS E2 USING (CIF)
        GROUP BY E2.NOMBRE
        ORDER BY EMPRESA;
